Join Behavior tab

The Process Join Behavior tab determines how the process will handle join failures. When set to Yes, any JoinFailure fault (detailed in the WS-BPEL Standard Faults section of the OASIS specification: http://docs.oasis-open.org/wsbpel/2.0/OS/wsbpel-v2.0-OS.html#_Toc164738543) will be ignored for all activities in the process. An activity is able override this value, or inherit the value from its parent.
